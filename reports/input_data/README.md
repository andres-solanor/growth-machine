# Input data

Place your cart-level sales export CSV files here.

The sales report auto-selects the **most recently modified** `.csv` in this folder when you omit `--input`.

Only `sales_carts_sample.csv` is tracked in git (synthetic demo data). Your real exports are ignored by git—add them locally as needed.

Required columns are documented in [../docs/sales-report.md](../docs/sales-report.md).
